# Supply-Chain-Inventory-Analysis
Supply Chain Inventory Analysis using SQL, Power BI and Python


## Project Overview
This project analyzes inventory and sales data to identify stock risks, high-performing products, and inventory optimization opportunities.

## Tools Used
- Excel
- SQL
- Power BI
- Python

## Business Questions
1. Which products sell the most?
2. Which products sell the least?
3. Which products have low inventory?
4. Which products are at risk of stockout?
5. Which product categories generate the highest sales?

## Project Status
Data collection and exploration in progress.
